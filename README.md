# assets-inject-webpack-plugin

Inject assets into any kind of template, using the core implementation of [gulp-inject](https://github.com/klei/gulp-inject).

## Usage

## List of compatible file template types:
* Jade/Pug `.jade/.pug`
* HTML `.html`
* JSX `.jsx`
* LESS `.less`
* SASS/SCSS `.sass/.scss`
